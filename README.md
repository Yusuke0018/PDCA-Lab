# 🧪 PDCA-Lab (HypoLab) - 仮説検証型習慣形成アプリ

PDCAサイクルを活用した仮説検証型の習慣形成支援ツール。仮説を立てて検証期間をシャッフルで決定し、日々の実践を記録しながら科学的に習慣を身につけていきます。

## 📱 デモ
[https://yusuke0018.github.io/PDCA-Lab/](https://yusuke0018.github.io/PDCA-Lab/)

## 🎯 概要

HypoLabは、日々の習慣や行動改善の仮説を立て、ゲーミフィケーション要素を取り入れながら楽しく検証できるWebアプリです。「〇〇すると△△になる」という仮説を立て、ランダムに決められた期間で検証することで、マンネリ化を防ぎながら継続的な習慣形成をサポートします。

## ✨ 主な機能

### 1. 🧪 仮説立案と検証
- **仮説ベースの習慣形成**: 期待する変化を明確にして検証
- **IF-THENルール**: トリガーと行動を結びつけて実行しやすく
- **朝の1行宣言**: 毎朝、期待する変化を宣言してモチベーション維持
- **複数仮説の同時進行**: 複数の習慣を並行して管理可能

### 2. 📅 柔軟な頻度設定
- **毎日実施**: 毎日取り組む習慣に（検証期間: 3-30日）
- **週◯回実施**: 週1-7回で無理のないペースを設定（検証期間: 2-10週）
- **特定曜日**: 月・水・金など、決まった曜日に実施（検証期間: 2-10週）

### 3. 🎲 シャッフルシステム
検証期間をランダムに決定することで新鮮さを維持：

#### 毎日実施の場合
- 短期間: 3〜7日
- 中期間: 8〜14日
- 長期間: 15〜30日

#### 週◯回・特定曜日の場合
- 短期間: 2〜4週（14〜28日）
- 中期間: 5〜7週（35〜49日）
- 長期間: 8〜10週（56〜70日）

### 4. 📊 進捗管理とカレンダー
- **ビジュアルカレンダー**: 
  - 達成日は緑色で表示
  - 未達成日は赤色で表示
  - 将来の日付はグレーアウト
  - 週ごとの達成状況バー（週◯回設定時）
  - 特定曜日は青枠で強調表示
- **自動計算**: 頻度設定に応じた正確な達成率計算
- **強度調整**: 日ごとに「軽め（×0.8）/基本（×1.0）/高強度（×1.2）」を設定可能
- **週の管理**: 検証開始日から7日ごとを1週間として計算

### 5. 🎴 カードシステム
達成率に応じてカードを獲得：

#### 報酬カード（高達成率で獲得）
- **スキップチケット**: 1日分の実施を免除（過去/今日に使用可）
- **達成ブースト**: 今日を含む3日を自動達成
- **達成率ブースター**: 最終達成率に+15%
- **パーフェクトボーナス**: 次回100%達成時にボーナスカード
- **モチベートカード**: 励ましのメッセージ

#### ペナルティカード（低達成率で獲得）
- **短期集中**: 次の仮説は短期間に固定
- **ハードモード**: 90%未満はカード獲得なし
- **リセットリスク**: 3日連続未達成で全リセット
- **期間延長**: 検証期間が延長される

### 6. 🎉 モチベーション機能
- **達成アニメーション**: 達成時に花火エフェクト表示
- **視覚的フィードバック**: 進捗に応じた色の変化
- **スワイプ操作**: モバイルで快適な画面遷移
- **削除機能**: 長押し（スマホ）/右クリック（PC）で仮説を削除

## 🎮 使い方

### 初回セットアップ
1. アプリにアクセス（またはホーム画面に追加）
2. 「➕ 新規仮説立案」をタップ
3. 仮説内容を入力：
   - **タイトル**: 例「毎日10分の瞑想」
   - **詳細説明**: 検証したい内容の詳細
   - **朝の1行宣言**: 期待する変化（例：集中力が上がる）
   - **IF-THENルール**: 任意（例：IF 朝起きたら THEN 瞑想する）

### 頻度と期間の設定
1. **実施頻度を選択**：
   - 毎日
   - 週◯回（スライダーで1〜7回を指定）
   - 特定の曜日（月〜日から複数選択可）

2. **検証期間を選択**：
   - 短期間 / 中期間 / 長期間
   - シャッフルボタンで具体的な日数/週数がランダムに決定

### 日々の記録
1. ホーム画面から進行中の仮説をタップ
2. 進捗カレンダーで今日の日付をタップして達成記録
3. 長押し（または右クリック）で強度を調整可能

### 朝の宣言
1. 進捗画面の「☀️ 朝の宣言」ボタンをタップ
2. 今日の意気込みや期待する変化を入力
3. 宣言済みの日はカレンダーに黄色のドットで表示

### カード使用
1. 進捗画面の「🎴 カードを使用」ボタンをタップ
2. 使用可能なカードから選択
3. スキップチケットは過去や今日の未達成日に適用可能

## 💡 活用例

### 運動習慣
- **仮説**: 「朝10分のストレッチで1日の集中力が上がる」
- **頻度**: 毎日
- **期間**: 短期間（3〜7日でテスト）
- **IF-THEN**: IF 起床したら THEN ストレッチマットを広げる

### 学習習慣
- **仮説**: 「週3回の英語学習で3ヶ月後にTOEIC100点アップ」
- **頻度**: 週3回
- **期間**: 長期間（8〜10週）
- **IF-THEN**: IF 夕食後 THEN 英語アプリを開く

### 生活改善
- **仮説**: 「平日の23時就寝で朝の目覚めが良くなる」
- **頻度**: 月〜金の特定曜日
- **期間**: 中期間（5〜7週）
- **IF-THEN**: IF 22:30になったら THEN スマホを充電器に置く

## 🛠️ 技術仕様

### 対応環境
- **デバイス**: スマートフォン、タブレット、PC
- **ブラウザ**: Chrome, Safari, Firefox, Edge（最新版推奨）

## ❓ よくある質問

**Q. データはどこに保存されますか？**
A. ブラウザのローカルストレージに保存されます。ブラウザのキャッシュクリアで消える可能性があるため、重要なデータは定期的にメモしておくことをお勧めします。

**Q. 複数デバイスで同期できますか？**
A. 現在はローカル保存のみのため同期機能はありません。今後の実装を検討中です。

**Q. 週◯回の「週」はいつから始まりますか？**
A. 検証開始日から7日間を1週間として計算します。月曜始まりではありません。

**Q. 仮説を削除するには？**
A. 仮説カードを長押し（スマホ）または右クリック（PC）で削除できます。
